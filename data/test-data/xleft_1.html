Review of SIM reader
By El Tombo on December 29, 2012
Size: ALL IN 1 Verified Purchase
I bought this to deal with my wife's old sim card from her phone that took a dunking. The main purpose for me is to get the stuff off her old sim card and onto her new phone. I'm only going to review the SIM card function of this as used on Windows XP.

So it looks like all the other card slots EXCEPT the sim card pop up as drives in "my computer". Hardware wise it works fine so far. But I deduct one star for the software hassle (detailed below). I'll update if I have problems with the other features or have any "glitches".

For the sim card reader function you need the "USIM Editor" software (from way back in 2007) which is included on that little CD that comes with it. This software might be OK to use in an emergency if you HAVE to find that number now. But it is pretty useless if you're trying to extract all the info from your contacts list to a new phone. The main problems with this software are:
1) It only shows one name and one phone number for each entry and that's it. No second numbers, no email addresses, no nothing else.
2) It only extracts the information to a ".usv" format which is a PAIN to work with.
Most modern phones use ".csv" or ".vcf" formats and I couldn't find any easy to use converters on the web. Before I found other SIM-reading software, I wound up writing a short python script to handle the conversion, but there's still the original problem #1 in that there is a lot of info that's just not there in the file.

Luckily through some googling, I found "Dekart SIM Manager" which I was able to download and install and use (through a laborious process which I will detail out in a comment to this review). You get a free 30 day trial for that software, so I'm good for a little while anyway.

4 stars because the product works when you get the software right.
